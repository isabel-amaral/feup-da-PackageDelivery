◆ scenery1()

Company::scenery1Results Company::scenery1 ( )

Finds best distribution among the available drivers for the normal shipping orders. Considers three different sorting criteria for the list of company drivers and normal shipping deliveries:

  • descending in weight
  • descending in volume
  • descending in weight+volume
    and, for each, finds best distribution by calling allocatePackages().
    Returns
    struct with several values from the best scenery 1 algorithm among the three.
